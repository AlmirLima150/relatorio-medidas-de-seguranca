# relatorio-medidas-de-seguranca
Atividade prática da formação AWS Cloud Practitioner Certification, apresentando um relatório sobre três medidas de segurança em aplicações na AWS: controle de acesso com IAM, proteção contra ataques web com AWS WAF e mitigação de ataques DDoS com AWS Shield. O projeto demonstra conceitos básicos de segurança em nuvem. 🚀
